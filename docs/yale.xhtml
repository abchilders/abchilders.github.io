Yale Romanization for Cantonese Typer

To use: Type your Yale words in the text box below. To generate tone marks, click the buttons above the text box, OR type a number after a letter to modify it:

  • Level tone: 1 or -
  • Rising tone: 2 or /
  • Falling tone: 3 or \

Other hints:

  • Type a tone character twice to "escape" it and get the regular character: --, //, \\, 11, 22, 33
  • Type a tone character three times to both add a tone mark to a letter and then type that regular character after: ---, ///, \\\, 111, 222, 333

Examples:

  • Level tone: type "ne1" or "ne-" to get "nē"
  • Rising tone: type "ga2m" or "ga/m" to get "gám"
  • Falling tone: type "m3h" or "m\h" to get "m̀h"
  • Tones and punctuation: type "ma-t--ye/h//me---ye/h" to get "māt-yéh/mē-yéh"
  • Numbers: type "11 -- me1" to get "1 - mē"
